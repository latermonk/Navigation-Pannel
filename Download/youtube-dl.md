

#  yt-dlp
* 按chapter分割   

```
yt-dlp --format "bestvideo[ext=mp4]+bestaudio[ext=m4a]"   --split-chapters    "p61x6y8VA2o"

```

* 下载某一个时间段儿   
```
yt-dlp --format "bestvideo[ext=mp4]+bestaudio[ext=m4a]" --no-mtime --output "%(title)s.%(ext)s" --download-sections "*0:00-0:07" 0gh3f9djxEQ
```
